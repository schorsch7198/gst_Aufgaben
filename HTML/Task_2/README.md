# HTML Task 2 - Tables

This task demonstrates HTML table structure and formatting.

## Files

- **table_staff.html** - Employee table with structured data

## Topics Covered

- Table structure (`<table>`, `<thead>`, `<tbody>`, `<tfoot>`)
- Table rows and cells (`<tr>`, `<th>`, `<td>`)
- Column grouping (`<colgroup>`, `<col>`)
- Table borders and width attributes
- Table headers and data organization
- Row and column spanning (`rowspan`, `colspan`)
