# HTML Task 1 - Basic HTML Page Structure

This task demonstrates fundamental HTML elements and their display on a web page.

## Files

- **page_blank.html** - Empty template to start from
- **page_complete.html** - Complete example with all HTML elements
- **val_test_page.html** - Validation test page

## Topics Covered

- Basic HTML document structure (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
- Headings (`<h1>` - `<h6>`)
- Paragraphs and text formatting (`<p>`, `<strong>`, `<em>`)
- Lists (ordered and unordered)
- Links and images
- Tables
- Basic inline CSS styling
