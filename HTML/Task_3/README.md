# HTML Task 3 - Forms

This task demonstrates comprehensive HTML form elements and input types.

## Files

- **formula.html** - Complete user registration form with various input types

## Topics Covered

- Form structure (`<form>`, `<fieldset>`, `<legend>`)
- Text inputs (`text`, `email`, `password`, `tel`, `url`)
- Number and date inputs (`number`, `date`, `time`)
- Selection elements (`<select>`, `<option>`, multiple select)
- Radio buttons and checkboxes
- Text areas (`<textarea>`)
- File uploads (`file`)
- Color picker (`color`)
- Range slider (`range`)
- Hidden inputs
- Form validation (required, pattern, min/max)
- Labels and placeholders
- Submit and reset buttons
